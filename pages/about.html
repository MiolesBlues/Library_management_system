Project Information

About This Project

LibraryMS is a simple web application for managing books, members, and loans through one clear and user-friendly interface.

Overview

What the system does

Purpose

Built for library record management

This project was created to demonstrate how a frontend website can work with a relational database using Supabase. It focuses on the core tasks needed in a small library system.

Main Features

Core parts of the project

  • manage book records
  • manage member records
  • issue and return books
  • track overdue loans
  • show dashboard summary information

Technology

Tools used in this project

  • HTML, CSS, and JavaScript for the frontend
  • Supabase and PostgreSQL for the database
  • GitHub for version control and teamwork

Current Pages

How the system is organised

  • Home shows dashboard summaries and quick links
  • Members supports CRUD actions and search
  • Loans supports issue, return, and overdue tracking
  • Books is prepared for book management work
  • ERD and SQL files explain the database structure
