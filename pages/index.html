Library Management System

A simple and structured web application for managing books, library members, and loans in one place. The system is designed for clear navigation, readable content, and accessible interaction.

Go to Books View Project Info

Purpose

The system helps manage books, members, and loan records through connected pages and a database-driven structure.

Technology

Built with HTML, CSS, JavaScript, Supabase, and GitHub for teamwork, version control, and database integration.

Accessibility

The interface uses clear headings, readable spacing, keyboard focus states, and a high contrast mode for users with low vision.

Dashboard Overview

This homepage will later display summary cards for total books, total members, active loans, and overdue books. It is currently prepared as the central navigation and overview page for the project.
